# Spring AI Tools 실습 가이드

## 1. 실습 목표

이 가이드는 Spring AI의 **Function Calling** 기능을 활용하여 LLM이 외부 도구(Java 메서드)를 호출하는 방법을 실습합니다.
`@Tool` 어노테이션을 사용해 도구를 등록하고, `ChatClient`가 대화 문맥에 따라 적절한 도구를 선택하여 실행하는 과정을 경험합니다.

## 2. 사전 준비

```bash
export OPENAI_API_KEY=sk-...

mvn clean install -DskipTests
mvn spring-boot:run
```

- 접속 URL: `http://localhost:8080`

## 3. 실습 시나리오

### 3.1 날짜/시간 도구 (DateTime Tools)
- **목표**: 가장 기본적인 도구 등록 및 호출 확인
- **파일**: `com.example.springai.datetime.DateTimeTools`
- **기능**:
  - `getCurrentDateTime()`: 현재 날짜와 시간 반환
  - `setAlarm(time)`: 특정 시간에 알람 설정 (Mock)
- **테스트**:
  - "지금 몇 시야?"
  - "오후 3시에 알람 맞춰줘"

### 3.2 날씨 도구 (Weather Tools)
- **목표**: 외부 API(OpenWeatherMap 등) 연동 도구 실습
- **파일**: `com.example.springai.weather.WeatherTools`
- **기능**:
  - `getWeather(city, country)`: 특정 도시의 날씨 정보 조회
- **테스트**:
  - "서울 날씨 어때?"
  - "도쿄의 현재 기온은?"

### 3.3 파일 시스템 도구 (File System Tools)
- **목표**: 로컬 리소스(파일) 접근 및 조작 도구 실습
- **파일**: `com.example.springai.filesystem.FileSystemTools`
- **기능**:
  - `saveQuestionToFile(text)`: 텍스트를 파일로 저장
  - `listFiles()`: 현재 디렉토리 파일 목록 조회
  - `deleteFile(fileName)`: 파일 삭제
- **테스트**:
  - "오늘 배운 내용을 파일로 저장해줘"
  - "파일 목록 보여줘"
  - "방금 만든 파일 삭제해줘"

### 3.4 음악 도구 (Music Tools)
- **목표**: 창의적인 도메인 도구 구현 및 복합 파라미터 처리
- **파일**: `com.example.springai.music.MusicTools`
- **기능**:
  - `findSong(title, artist)`: 노래 검색
  - `addToPlaylist(songId, playlistName)`: 플레이리스트 추가
- **테스트**:
  - "아이유의 좋은 날 찾아줘"
  - "그 노래 내 플레이리스트에 넣어줘"

## 4. 테스트 방법

### 웹 UI 활용
1. 브라우저에서 `http://localhost:8080` 접속
2. 상단 드롭다운에서 테스트할 도구 선택 (예: `/chat/date-time-tools`)
3. 메시지 입력 후 'Send' 버튼 클릭
4. LLM의 응답과 도구 호출 결과 확인

### CURL 활용
```bash
curl -X POST http://localhost:8080/chat/date-time-tools -d "message=지금 몇 시야?"
```
